Este seria o meu projeto de ADRC, com o prosseguimento do trabalho eu observei que não era possivel usar o smb como eu estava pensando,
assim perdendo totalmente o sentido do projeto.

1. INTRODUÇÃO:
   - para realizar esse experimento, você só precisa do seu console com um custom firmware(hen ou cfw) e necessita que ele esteja com
   - as opções de compartilhamento FTP ativas. E de um segundo aparelho para realizar a conexão FTP

2. REALIZAÇÃO:
   -Após isso você só precisa conectar um gerenciador de arquivos com o seu ps3 usando o ip do console e acessando como usúario para livre
   acesso, recomendo a utilização dos scripts python deste repositório para relizar os testes.
